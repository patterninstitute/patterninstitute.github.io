Scientific Open Source Software

The Pattern Institute maintains a curated listing of scientific open source software developed and supported by its research community. This collection includes R packages, data packages and reproducible compendia designed to advance computational biology, genomics, and data science. Each software entry is openly available, documented, and actively maintained to promote reproducibility, collaboration, and innovation in scientific research.
